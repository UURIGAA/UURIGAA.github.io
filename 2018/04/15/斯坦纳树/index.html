斯坦纳树

问题

在OI中我们经常会遇到这样一类问题：

给一张图以及一个点集$S$（点集大小很小），图中每条边有边权（边权为正）。要求在图中选出若干条边，使得$S $中的点连通，并且边权和最小。

这种问题就是斯坦纳树问题。我们要求的就是最小斯坦纳树。

解法

首先，最优解中的边一定构成了一棵树，否则把环删掉一条边答案一定会变小。

由于点集大小很小，考虑状压$DP$。

设状态$f[i][j]$表示：

以$i$为根且至少包含了集合$j$中的点的某个连通子树的最小边权和

其中$j$是$S$的一个子集。

有两种方式转移：

  • $f[i][j]=\mathrm{min}(f[i][j],f[i][s]+f[i][j-s])$ ($s$是$j$的子集)
  • $f[i][j]=\mathrm{min}(f[i][j],f[i’][j]+w[i][i’])$（$w[i][i’]$是$i$和$i’$之间的边长）

第一种转移是将不同子树的答案合并，第二种转移是在原树的基础上加进来一个点。

对于第一种转移，直接枚举子集。对于第二种转移，跑最短路即可。

最终答案就是$\mathrm{min}\{f[i][S]\}$.

例题

bzoj4774: 修路

Description

村子间的小路年久失修，为了保障村子之间的往来，法珞决定带领大家修路。对于边带权的无向图 G = (V, E)，请选择一些边，使得1 <= i <= d， i号节点和 n - i + 1 号节点可以通过选中的边连通，最小化选中的所有边的权值和。

Input

第一行两个整数 n, m，表示图的点数和边数。接下来的 m行，每行三个整数 ui, vi, wi，表示有一条 ui 与 vi 之间，权值为 wi 的无向边。

1 <= d <= 4

2d <= n <= 10^4

0 <= m <= 10^4

1 <= ui, vi <= n

1 <= wi <= 1000

Output

一行一个整数，表示答案，如果无解输出-1.

题解

斯坦纳树模板题。

设状态$f[i][j]$表示：以$i$为根且至少包含了集合$j$中的点的某个连通子树的最小边权和，其中$j$是$S$的一个子集。

$g[j]$表示考虑了点集$j$中的点的最优选边方案的边权和，其中$j$是$S$的一个子集。

我们判定$g[j]$合法当且仅当要连通的点对同时出现或同时不出现在状态$j$中。

那么$g[j]=\mathrm{min}\{g[j],g[s]+g[j-s]\}$，且$s$是合法状态。

这样我们先用斯坦纳树的方法求出$f$，从而得到初始的$g$.

显然$g[j]=\mathrm{min}\{f[i][j]\}$

然后枚举当前状态$j$以及当前状态的合法子集更新$g$.

最终$g[S]$即为答案（$S$是题中给出的点集）。

bzoj4006: [JLOI2015]管道连接

Description

小铭铭最近进入了某情报部门，该部门正在被如何建立安全的通道连接困扰。

该部门有 n个情报站，用 1 到 n 的整数编号。给出 m对情报站 ui,vi和费用 wi，表示情报站 ui和 vi之间可以花费 wi单位资源建立通道。

如果一个情报站经过若干个建立好的通道可以到达另外一个情报站，那么这两个情报站就建立了通道连接。形式化地，若 ui和 vi建立了通道，那么它们建立了通道连接；若 ui和 vi均与 ti建立了通道连接，那么 ui和 vi也建立了通道连接。现在在所有的情报站中，有 p 个重要情报站，其中每个情报站有一个特定的频道。

小铭铭面临的问题是，需要花费最少的资源，使得任意相同频道的情报站之间都建立通道连接。

Input

第一行包含三个整数 n,m,p，表示情报站的数量，可以建立的通道数量和重要情报站的数量。

接下来 m 行，每行包含三个整数 ui,vi,wi，表示可以建立的通道。最后有 p 行，每行包含两个整数 ci,di，表示重要情报站的频道和情报站的编号。

Output

输出一行一个整数，表示任意相同频道的情报站之间都建立通道连接所花费的最少资源总量。

Sample Output

样例解释

选择 (1,5); (3,5); (2,5); (4,5)这四对情报站连接。

数据范围与提示

对于 100% 的数据，0<ci≤p≤10, 0<ui,vi,di≤n≤1000, 0≤m≤3000, 0≤wi≤200000。
